# Csomagszállító Szimulátor (Console App)

Ez egy C# nyelven íródott konzolos alkalmazás, amely egy logisztikai csomagszállító rendszer alapvető működését szimulálja. A projekt célja a futárok, csomagok és szállítási útvonalak kezelésének bemutatása objektumorientált környezetben.

## Funkciók

* **Csomagkezelés:** Új csomagok felvétele a rendszerbe egyedi azonosítóval.
* **Futár menedzsment:** Futárok hozzárendelése a szállításokhoz.
* **Állapotkövetés:** A csomagok aktuális státuszának (pl. raktárban, szállítás alatt, kézbesítve) nyomon követése.
* **Szimulációs naplózás:** A konzolon követhető események a szállítási folyamat során.

## Technológiák

* **Nyelv:** C#
* **Keretrendszer:** .NET 10.0.x (Console Application)
* **Paradigma:** Objektumorientált programozás (OOP)

## Telepítés és Futtatás

A futtatáshoz szükséges a [.NET SDK](https://dotnet.microsoft.com/download).
